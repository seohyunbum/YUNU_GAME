# 1인칭 야생 마을 보스 챕터 / 경제 밸런스 표

작성일: 2026-06-06  
목적: 최종 보스 처치까지의 장기 목표를 명확히 만들고, 미니게임 포인트/재료/거래/장비 제작의 경제 흐름을 한눈에 조정할 수 있게 한다.

## 기준

- 현재 구현된 보스 순서: 용 -> 파이어 드래곤 -> 레드 드래곤 -> 레이저 드래곤 -> 다크 드래곤 -> 불멸의 존재
- 현재 레벨 산식: 다음 레벨 필요 경험치 = `floor(45 * 현재레벨^1.35)`
- 레벨 1당 상승: 최대 체력 +1, 방어력 +1, 공격력 +1
- 현재 보스 전리품: 용의 비늘 50%, 용의 꼬리 40%, 용의 뿔 10%
- 현재 포인트 교환: 용의 비늘 4000P, 용의 꼬리 5000P, 용의 뿔 10000P
- 현재 보스들은 월드에 배치되어 있으며, 아래 표의 "권장 개방 조건"은 다음 구현 단계에서 게이트로 연결할 기준이다.
