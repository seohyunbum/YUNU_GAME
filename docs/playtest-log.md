# 플레이테스트 로그

> 목적: 자동 테스트(verify/visual-check)가 **못 잡는 것** — 손맛, 난이도 체감, 재미 — 을 사람이 직접 확인하고 기록한다.
> 사용법: 한 세션 끝날 때마다 아래 양식을 복사해 맨 위에 추가. 짧아도 좋으니 **그날 느낀 것**을 남긴다 (연우의 한 줄 감상 환영).

## 우선 확인 체크리스트 (최근 추가 기능)

플레이 중 자연스럽게 아래를 지나가도록 동선을 잡는다. 전부 한 세션에 할 필요 없음.

### 진행/보스
- [ ] 챕터 보스가 권장 레벨 맵에 실제로 있다 (드래곤 평원 Lv20 → 버섯 숲 Lv40 → 독 늪 Lv55 → 산맥 Lv75 → 설원 Lv100 → 용의 땅 Lv130)
- [ ] 지도(M)에 챕터 보스·필드 보스 마커가 보이고, 봉인된 보스는 봉인 표시가 있다
- [ ] 필드 보스 처치 → 미니 팡파레 → 같은 맵에 다시 안 나타난다
- [ ] 불멸의 존재 처치 → 폭죽 + 팡파레 + 크레딧 엔딩
- [ ] HUD 권장 레벨/목표 문구가 실제 위치와 일치한다

### 전투 체감
- [ ] 근접 몬스터 공격 모션이 눈에 띈다 (움찔→돌진이 구분됨)
- [ ] 인벤토리/제작대/지도 패널 열린 동안 몬스터에게 안 맞는다
- [ ] 고레벨 캐릭터도 보스급에게 데미지가 들어온다 (무적 아님 — 15% 하한)

### 세이브
- [ ] 저장 슬롯 5개까지 생성, 꽉 차면 덮어쓸 슬롯 선택 UI
- [ ] 불러오기 패널 "파일로 백업" → JSON 다운로드 / "파일 가져오기" → 그 시점 복원
- [ ] 제작대 회수 후에도 완료된 퀘스트가 풀리지 않는다

### 수리 (신규)
- [ ] 닳은 도끼/곡괭이/삽이 있으면 제작대에 "도구 수리" 섹션이 뜬다
- [ ] 제련 재료 1개로 내구도 50% 회복, 재료 없으면 버튼 비활성
- [ ] 내구도 0 도달 시 여전히 부서진다 (수리는 예방)

### 튜토리얼/초반
- [ ] 새 캐릭터로 23단계 튜토리얼이 자연스럽게 이어진다
- [ ] 직업별 무기 제작 퀘스트의 재료·레시피가 실제 제작과 일치한다
- [ ] 공동묘지 초록손 무덤(1/15) → 지하 좀비전 → 승리 시 탈출

## 버그 기록 양식

심각도: **막힘**(진행 불가) / **거슬림**(불편하지만 진행 가능) / **사소**(겉보기)
