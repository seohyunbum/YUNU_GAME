# 3D 야생 게임 구현 노트

## 핵심 방향

- 시점은 1인칭 전용입니다.
- 화면 중앙 조준점으로 대상을 바라보고 `E`를 눌러 상호작용합니다.
- 초반 목표는 작은 나무, 상자, 망치, 제작대, 제련대, 도구 제작 흐름을 익히는 것입니다.
- 구조물과 생물은 통과할 수 없고, 낮은 오브젝트는 점프로 넘을 수 있습니다.
- 동굴은 야생과 다른 어두운 분위기와 둥근 돌벽으로 구성됩니다.
- 새로시작, 저장, 불러오기는 화면 왼쪽 위 버튼과 단축키로 사용할 수 있습니다.

## 현재 구현된 플레이 흐름

1. 튜토리얼 책을 읽습니다.
2. 작은 나무를 캐서 나무를 얻습니다.
3. 100걸음마다 생길 수 있는 상자를 찾아 망치를 얻습니다.
4. 인벤토리의 2x2 미니 제작대에 나무 4개를 넣고 제작대를 만듭니다.
5. 제작대를 `P`로 설치하고 `E`로 열어 도구, 무기, 갑옷, 가방을 만듭니다.
6. 동굴에서 돌, 석탄, 흑요석을 찾고 제련대를 이용해 더 강한 장비를 만듭니다.
7. 흙 지역은 손이나 삽으로 파고, 돌층이 나오면 더 이상 팔 수 없습니다.
8. 마을 식량창고에서 고기를 가져오면 기사들이 공격합니다.
9. `Ctrl+S`로 저장하고 `Ctrl+L`로 불러옵니다. `N`은 새로시작입니다.
