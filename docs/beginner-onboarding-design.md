# 초보 온보딩 고도화 — 제작대·가방을 절대 놓치지 않게

작성 2026-06-21. 목표: **이 게임을 처음 켠 사람**이 (사실상 필수인) **제작대 설치 → 가방 제작**까지 막힘 없이 손에 잡히듯 따라오게 만든다.

## 0. 현재 동작 (코드 근거)

| 항목 | 현재 | 근거 |

| 핫키 | I=가방, Q=퀘스트/보상, E=줍기·대화·진입, 우클릭=사용·먹기 | `ui/controlsGuide.ts` |

핵심: 메커니즘·퀘스트·온보딩 UI는 **이미 다 있다.** 빠진 건 **초보 발견성(discoverability)** 한 겹이다.

## 1. 진짜 초보가 막히는 지점 (가설)

1. **상세를 안 본다.** HUD엔 제목만 보이고 "어떻게"는 hover/클릭 뒤. 초보는 카드를 안 누른다 → 뭘 눌러야 할지 모른 채 헤맨다.
2. **I키를 모른다.** 좌측 조작표에 있으나 밀도가 높아 놓친다. "제작대 만들기" 퀘스트를 받고도 인벤 여는 법을 못 찾는다.
3. **제작대는 '설치'가 필요.** 핫바에서 쓰려다 실패. 우클릭-설치가 비직관(다른 게임은 들고 월드에 설치).
4. **2x2 vs 3x3 혼동.** 이미 2x2가 있는데 제작대를 왜 또? 어디서 만드나?
5. **가방 작음 → 루팅 거부 벽.** ⚠️에 부딪힐 때까지 가방 필요성을 모르고, 경고를 봐도 "가죽 7→가방" 행동으로 연결이 안 된다.

## 2. 설계 원칙

- **푸시 > 풀.** 결정적 순간엔 안내를 **먼저** 띄운다(클릭을 기다리지 않음). 단, **1회·해제 가능**.
- **다음 행동 1개.** 한 번에 "지금 누를 키 1개"만 강조.
- **새 시스템 X, 기존 흐름 또렷이.** 별도 튜토리얼 모드가 아니라 기존 퀘스트를 선명하게.
- **저위험·가역 우선.** 텍스트·1회 팝업·뱃지부터. UI 개편은 분리해 나중에.
- **휘발 상태.** "이미 본 안내" 기록은 메모리 Set(세이브 무관) → 회귀·소급 리스크 0.

## 3. 구현안 (3티어, 누적)

### Tier 1 — 가이드·문구·뱃지 강화 (저위험, 권장 시작)

- **1A. 핵심 스텝 1회 자동 가이드.** `craft_workbench_item` / `place_workbench` / `craft_bag` 스텝이 **활성으로 바뀌는 순간** `objectiveGuide`를 **1회 자동** 표시(본 스텝 id는 휘발 Set 기록). → 클릭 안 해도 "어떻게"를 본다.
- **1B. 첫 제작대 획득 힌트.** `crafting_table`을 처음 인벤에 넣는 순간 "🛠️ **I**로 가방을 열고 제작대를 **우클릭**해 바닥에 설치하세요"(1회).
- **1C. 첫 인벤-풀 가방 가이드.** 가방이 처음 가득 차 줍기 거부될 때, 기존 ⚠️ + `craft_bag` 가이드 1회 자동.
- **1D. 퀘스트 문구 단계번호화.** 제작대·가방 스텝을 "① I 열기 → ② 2x2에 나무3+망치1 → ③ 제작 → ④ 우클릭 설치" 식 절차로.
- **1E. 인벤 패널 placeable 뱃지.** 설치형 아이템 슬롯에 "우클릭=설치" 작은 뱃지/테두리.

### Tier 2 — 첫 챕터 코치 비콘 (중간)

- HUD에 **1챕터 한정 "코치" 스트립**: 현재 스텝의 **다음 행동 키**를 큼직하게(예: `[I] 가방 열기`). 스텝 진행 시 자동 갱신, X로 해제. 키 칩 펄스.
- 눌러야 할 UI(예: 가방/핫바)에 화살표·펄스 스포트라이트(설치·가방 단계 한정).

### Tier 3 — 인벤토리 UI 개편 (큼, 옵션)

- "미니 제작대 2x2(어디서나)" vs "제작대 3x3(설치 후)" **시각 구분 + 한 줄 설명**.
- placeable **우클릭-설치 어포던스 시각화**(드롭존 강조·아이콘).
- **상시 "가방 n/8칸"** 표시; 6/8↑ 주황 "곧 가득—가죽 7로 가방 제작" 인라인 바.
- 제작 결과 미리보기·재료 부족 하이라이트.

## 4. 권장 실행 순서

**Tier 1 먼저 구현·배포**(초보 막힘의 ~80% 해소, 저위험). 반응 보고 Tier 2(코치 비콘) → 필요 시 Tier 3(UI 개편).

## 5. 검증 계획

- E2E(새 게임): 각 핵심 스텝 활성 시 가이드 1회 자동 노출 / 첫 제작대 힌트 / 첫 인벤풀 가이드 / placeable 뱃지 존재.
- 회귀: 기존 퀘스트 진행·보상·세이브 소급(자동 팝업은 휘발 Set이라 세이브 무관), main.ts 예산/메서드 게이트.
