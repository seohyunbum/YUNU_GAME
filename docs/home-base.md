# 내 집 베이스캠프 (Home Base)

> 상태: **구현 완료** (2026-06-11).
> 관련 코드: `src/game/homeBase.ts` (창고/보급 로직) · `src/game/interiors.ts` (내 집 가구) · `src/ui/homeStoragePanel.ts` (창고 UI) · `src/main.ts` `claimHomeSupply`/`renderHomeStoragePanel`/사망 부활 분기.

## 1. 목적
집은 재료가 많이 드는데(통나무집 = 나무 120 + 막대 60 + 돌 40 + 망치 2) 보상이 일회성 상자 하나뿐이었다.
집을 **"고레벨 맵에 가기 전 정비하고, 죽으면 돌아오고, 주기적으로 보상을 받는 내 전용 베이스캠프"** 로 만든다.

## 2. 혜택 6종 — 플레이어가 지은 집(`playerOwned`)에만 적용

| 기능 | 구현 | 비고 |
|---|---|---|
| 안전지대 | 집 내부는 별도 인테리어 공간 — 몬스터 AI 는 overworld 에서만 동작(`predatorAi` 가드) | 입장 메시지로 명시 + 테스트로 고정 |
| 휴식 강화 | 내 침대(회수 불가)에서 E/우클릭 → **체력 완전 + 마나 완전 + 배고픔 1 회복** | 일반 설치 침대(체력만)보다 강함 |
| 귀환 지점 | 사망 시 내 집이 현재 맵에 있으면 confirm 으로 **집 앞 부활** 선택 (취소 = 마을) | 떨군 아이템 위치는 그대로 |
| 개인 보관함 | "집 창고" 24칸, 저장 파일에 영구 보존 (`player.homeStorage`) | 칸 클릭으로 인벤토리와 왕복 |
| 보급 상자 | **플레이 30분(1800초)마다 1회**. 레벨 구간별 보상 + 15% 보너스 1줄 | 쿨다운은 잔여 초로 저장 — 게임 시계가 하루 단위로 순환하기 때문 |
| 지도 표시 | 지역 지도에 🏠(초록 지붕) 마커 + 집 이름 | `[data-home-marker]` |

마을 NPC 집·대장간은 기존 그대로(일회성 상자 등) — 혜택 없음.

## 3. 보급 보상 테이블 (`rollHomeSupply`)

티어 = 레벨 1~14 / 15~29 / 30~54 / 55~84 / 85+.
기본: 고기 2+티어, 구급상자 1(티어1+: 2), 나무 20+10×티어, 돌 10+6×티어.
티어1+: 구리 / 티어2+: 철 / 티어3+: 금 3 / 티어4: 다이아 2 + 흑요석 2.
보너스(15%): 다이아 1(티어2+) 또는 금 1.
인벤토리가 가득 차 한 개도 못 받으면 쿨다운을 소모하지 않는다.

## 4. 저장 (SAVE_VERSION 10)
- `player.homeStorage: Slot[]` (24칸, 기본 빈 칸) / `player.homeSupplyCooldownSeconds` (기본 0 = 업그레이드 직후 즉시 1회 보급) / `player.currentHouseOwned` (집 안에서 저장 시 인테리어 복원용)
- `SavedObject.playerOwned` — 집 오브젝트에 직렬화.
- 마이그레이션·왕복 테스트: `save-migration-test` / `save-roundtrip-test`.

## 5. 미채택 대안
- **집 근처 야외까지 안전지대 확장**: 몬스터 도망/낚시 악용 여지 + 필드 긴장 유지 위해 내부만 안전.
- **보급을 게임 날짜 기준으로**: `worldTimeSeconds` 가 `% DAY_LENGTH` 로 순환해 누적 일수가 없음 → 플레이 시간 쿨다운 채택.
- **사망 시 무조건 집 부활**: 떨군 아이템이 마을 쪽일 수 있어 선택(confirm)으로.
